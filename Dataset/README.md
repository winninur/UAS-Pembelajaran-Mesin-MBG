# Dataset

Folder ini berisi seluruh dataset yang digunakan pada penelitian mengenai analisis topik percakapan masyarakat terhadap program **Makan Bergizi Gratis (MBG)** di media sosial X (Twitter).

Dataset terdiri atas data mentah hasil proses crawling serta data yang telah melalui tahapan preprocessing sehingga siap digunakan pada proses analisis.

## Isi Folder

- **MBG_2026.xlsx**  
  Dataset mentah (raw dataset) hasil proses crawling tweet menggunakan Tweet Harvest berdasarkan kata kunci yang telah ditentukan.

- **cleaned_text.csv**  
  Dataset hasil preprocessing yang telah melalui proses data cleaning, case folding, tokenisasi, stopword removal, dan stemming sehingga siap digunakan pada proses Topic Modeling dan Clustering.
